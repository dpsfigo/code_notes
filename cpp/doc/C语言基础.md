## C语言特点

### 优点

1. 代码量小

2. 执行速度快

3. 功能强大

4. 编程自由

 

### 缺点

1. 写代码实现周期长

2. 可移植性较差

3. 过于自由，经验不足易出错

4. 对平台库依赖较多
